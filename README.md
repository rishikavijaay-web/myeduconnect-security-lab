# MyEduConnect Security Lab

This repository contains the vulnerable MyEduConnect web application developed for the CCS6324 Ethical Hacking and Penetration Testing final assignment.

## Stack
- Ubuntu Desktop ARM64
- Apache2
- PHP
- MySQL
- UTM virtualized environment

## Main Features
- User registration
- Login
- Student profile
- Course browsing
- Course enrolment
- Mock payment workflow
- Admin panel
- REST API endpoint
- Comments page

## Deliberate Vulnerabilities
- SQL Injection
- Stored Cross-Site Scripting
- IDOR
- Broken Authentication
- Plaintext password storage
- Missing HTTPS
- Weak file permissions
- Directory listing
